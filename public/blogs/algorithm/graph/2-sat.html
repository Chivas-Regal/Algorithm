    2-SAT问题

    Chivas-Regal 建图、Kosaraju

    导引： AAA, BBB, CCC 三人中有两个女生
    如果 AAA 是男生，那么 BBB 一定是女生：
    AAA 和 CCC 性别相同
    求 AAA BBB CCC 三人的性别

    # 2 - SAT问题

    # 定义

    要确定N个bool类型的值，使得其满足所有的限制关系（每个限制关系最多只对两个bool变量进行限制）

    2-SAT问题是有多项式解法的，而3-SAT问题就是npc问题

    # 判断是否有解

    建立2N个点的有向图，不妨设
    iii 点表示 x[i]=×x[i] = \timesx[i]=×
    i′i'i′ 点表示 x[i]=✓x[i] = \checkmarkx[i]=✓
    一条 u→vu \rightarrow vu→v 的有向边表示如果选择了 uuu ，那么一定要选择 vvv
    不能出现 i↔i′i \leftrightarrow i'i↔i′
    判断i和i'是否位于同一强连通分量即可

    # 输出可行方案

    如果从 iii 点出发可以到达 i′i'i′ 点，则旋律i会导致选 i′i'i′ ，矛盾，因此只能选 i′i'i′
    Kosaraju算法在求强连通分量时顺带求出了一个遍历顺序 q[1],q[2],...,q[cnt]q[1], q[2], ..., q[cnt]q[1],q[2],...,q[cnt]
    只需要比较i和i'所属强连通分量在q中位置的前后关系即可构造出可行方案

    # 一元限制的构图方案

    限制： x[i]=✓x[i] = \checkmarkx[i]=✓
    连边： i→i′i \rightarrow i'i→i′
    表示如果 iii 选了 ×\times× 那么 iii 必须要是 ✓\checkmark✓

    # 二元限制的构图方案

    一条边 u→vu \rightarrow vu→v 的含义：若 uuu 则 vvv
    命题”若 uuu 则 vvv “的逆否命题为”若非 vvv 则非 uuu “
    命题于逆否命题等价
    所以在建 u→vu \rightarrow vu→v , 也要建 v→u′v \rightarrow u'v→u′
    所以对于每条二元限制，将其对应的命题与逆否命题的边连上，缺一不可

    建图例：
    限制： u∣v=✓u | v = \checkmarku∣v=✓
    （有一个满足 ✓\checkmark✓ 就行） u→v′,v→u′u\rightarrow v', v\rightarrow u'u→v′,v→u′

    建图例：
    限制： u∣v=×u | v = \timesu∣v=×
    （两者都要 ×\times× 回到一元限制关系） u′→u,v→v′u'\rightarrow u, v\rightarrow v'u′→u,v→v′

    建图例：
    限制： u&v=×u \And v = \timesu&v=×
    （两个没有同时满足） u′→v,v′→uu'\rightarrow v, v'\rightarrow uu′→v,v′→u

    建图例：
    限制： u&v=✓u \And v = \checkmarku&v=✓
    （两个都要满足） u′→u,v′→vu'\rightarrow u, v'\rightarrow vu′→u,v′→v

    建图例：
    限制： u≠vu \neq vu=v
    （两个情况不同） u′→v,u→v′,v′→u,v→u′u'\rightarrow v, u\rightarrow v', v'\rightarrow u, v\rightarrow u'u′→v,u→v′,v′→u,v→u′

    建图例：
    限制： u=vu = vu=v
    （两个情况相同） u′→v′,u→v,v′→u′,v→uu'\rightarrow v', u\rightarrow v, v'\rightarrow u', v\rightarrow uu′→v′,u→v,v′→u′,v→u

    # 例题

    有 nnn 对夫妻被邀请参加一个聚会，因为场地的问题，每对夫妻中只有 111 人可以列席
    在 2n2n2n 个人中，某些人之间有着很大矛盾（夫妻之间没有），主办方希望有矛盾的两个人不能同时出现在聚会上
    问有没有可能回有 nnn 个人同时列席？
    思路：
    建立 2n2n2n 个点的图，i点表示第i对夫妻是丈夫到场， i′i'i′ 点表示妻子到场
    如果 AAA 和 BBB 矛盾，则可以表示”若 AAA 则非 BBB “、”若 BBB 则非 AAA “。

    # 强连通分量求出后的处理

    bool res = 1;
for(int i = 0; i < N; i ++){
    if(nod[i] == nod[trans(i)]){
        res = 0;
        break;
    }
}
if(res) cout << "YES" << endl;
else    cout << "NO"  << endl;

    # 例题

    TTT 个 333 人队伍，每个队伍有一个队长
    每个队伍要么队长回家，剩下两个留校，要么队长留校，剩下两人回家
    给定 MMM 条限制，每条限制指定两个人 AAA 和 BBB ，要求 AAA 和 BBB 不能同时留校
    判断是否有合法方案
    思路：
    建立 6T6T6T 个点的有向图， iii 点表示第 iii 个人留校， i′i'i′ 点表示第 iii 个人回家
    对于每个队伍 xxx , yyy , zzz ，其中 xxx 为队长
    xxx 和 yyy 的选择不同，按照 x≠yx\neq yx=y 连边
    xxx 和 zzz 的选择不同，按照 x≠zx\neq zx=z 连边
    yyy 和 zzz 的选择不同，按照 y=zy=zy=z 连边
    对于每条限制 AAA ，BBB ，两人不能同时留校
    若 AAA 留校，则 BBB 回家： A→B′A\rightarrow B'A→B′
    若 BBB 留校，则 AAA 回家： B→A′B\rightarrow A'B→A′
