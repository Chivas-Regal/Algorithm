    vector<int> prime;
bool isprime[N];
inline void GetPrime(){
	isprime[0] = isprime[1] = 1;
	for(int i = 2; i < N; i ++){
		if(!isprime[i]) prime.push_back(i); //若未被标记，则i为质数
		for(int j = 0; j < prime.size() && prime[j] * i < N; j ++){
			isprime[i * prime[j]] = 1; //打上合数标记
			if(i % prime[j] == 0) break; //p[j]为i的约数时退出
		}
	}
}

    # 算术基本定理

    算术基本定理，又称唯一分解定理
    定理内容是：
    任何一个大于111的自然数NNN，如果NNN不为质数，都可以唯一分解成有限个质数的乘积 N=P1a1P2a2...PnanN=P_1^{a_1}P_2^{a_2}...P_n^{a_n}N=P1a1​​P2a2​​...Pnan​​ 。这里 P1<P2<...<PnP_1\lt P_2\lt...\lt P_nP1​<P2​<...<Pn​ 均为质数，指数 aia_iai​ 均为正整数
    这样的分解称为NNN的标准分解式

    在线性筛中的应用：每个数因它最小的质数而被标记
    规则体现在：如果某个数numnumnum包含了某个divdivdiv，那么这个最小divdivdiv可以和别的数凑成一个更大的数去标记，那么此时后面就不需要继续用numnumnum去凑数标记了。

    例：$(i,j)=(4,2),\quad ii%j=0i，那么说明444中有222，走到后面有个121212，121212可以因为(6,2)(6,2)(6,2)被标记，那么此时就不需要用(4,3)(4,3)(4,3)去进行标记了

    # 欧拉函数

    # 概念

    ∑i=1n[gcd(i,n)=1]\sum\limits_{i=1}^n[gcd(i,n)=1]i=1∑n​[gcd(i,n)=1] ,记为 ϕ(n)\phi(n)ϕ(n)
    欧拉函数的通式： ϕ(x)=x∗∏i=1n(1−1pi)\phi(x)=x*\prod\limits_{i=1}^n(1-\frac1{p_i})ϕ(x)=x∗i=1∏n​(1−pi​1​)
    欧拉函数的性质： nnn的因子的欧拉函数加起来等于nnn

    # 证明

    情况1.如果 nnn 是一个素数:
    n=pϕ(n)=n(1−1p)=p(1−1p)=p−1n=p\quad\phi(n)=n(1-\frac1p)=p(1-\frac1p)=p-1n=pϕ(n)=n(1−p1​)=p(1−p1​)=p−1

    情况2.如果 nnn 是一个素数 ppp 的 α\alphaα 次幂:
    那么从1→n1\rightarrow n1→n中，只有ppp的倍数不与nnn互质，从111到nnn， p、2p、3p、.....、pα−1pp、2p、3p、.....、p^{\alpha-1}pp、2p、3p、.....、pα−1p 一共有 pα−1p^{\alpha-1}pα−1 个
    所以 ϕ(n)=n−pα−1=pα−pα−1=pα(1−1p)=n(1−1p)\phi(n)=n-p^{\alpha-1}=p^{\alpha}-p^{\alpha-1}=p^{\alpha}(1-\frac1p)=n(1-\frac1p)ϕ(n)=n−pα−1=pα−pα−1=pα(1−p1​)=n(1−p1​)

    情况3.如果 nnn 是两个素数 ppp qqq 的乘积:
    显然此时与nnn互质的数既不是 ppp 的倍数也不是 qqq 的倍数。类似的情况222中的分析，ppp的倍数应有qqq个，qqq 的倍数有ppp个，但是还多算了一个既是ppp也是qqq的倍数nnn，于是 ϕ(n)=n−(p+q−1)=p∗q−p−q+1=(p−1)(q−1)=n(1−1p)(1−1q)\phi(n)=n-(p+q-1)=p* q-p-q+1=(p-1)(q-1)=n(1-\frac1p)(1-\frac1q)ϕ(n)=n−(p+q−1)=p∗q−p−q+1=(p−1)(q−1)=n(1−p1​)(1−q1​)
    还可以发: ϕ(n)=ϕ(p×q)(1−1p)(1−1q)=ϕ(p)×ϕ(q)\phi(n)=\phi(p\times q)(1-\frac 1p)(1-\frac 1q) = \phi(p)\times\phi(q)ϕ(n)=ϕ(p×q)(1−p1​)(1−q1​)=ϕ(p)×ϕ(q)

    概念：积性函数和完全积性函数 (欧拉函数是积性函数)
    完全积性函数：ϕ(n)=ϕ(p×q)=ϕ(p)×ϕ(q)\phi(n)=\phi(p\times q)=\phi(p)\times\phi(q)ϕ(n)=ϕ(p×q)=ϕ(p)×ϕ(q) 中不要求 ppp qqq 互质
    积性函数：ϕ(n)=ϕ(p×q)=ϕ(p)×ϕ(q)\phi(n)=\phi(p\times q)=\phi(p)\times \phi(q)ϕ(n)=ϕ(p×q)=ϕ(p)×ϕ(q) 中要求 ppp qqq 互质

    情况4.nnn 是某个 mmm 和一个于 mmm 互质的素数 ppp 的乘积，即: n=m∗pn = m * pn=m∗p：（完整的证明需要各种定理，这里不详解）假设 mmm 是之前讨论的情况，即 mmm 满足欧拉函数公式： $$\phi(m)=m*\prod^k_{i=1}(1-\frac1{p_i})$$ 同时因为 n=m∗pn = m * pn=m∗p， mmm 与 ppp 互质， 则有： $$\phi(n)=\phi(mp)=\phi(m)\phi(p)$$ 另外，由情况111可知： ϕ(p)=p−1=p(1−1p)\phi(p)=p-1=p(1-\frac1p)ϕ(p)=p−1=p(1−p1​)
    显然， ϕ(m)∗ϕ(p)\phi(m) * \phi(p)ϕ(m)∗ϕ(p) 得到的公式，依然满足欧拉函数

    情况5. n=m∗pn = m * pn=m∗p， ppp 是素数，但是 ppp 不与 mmm 互质
    ppp 是素数，但不与 mmm 互质，显然 ppp 是 mmm 的因子，可以转化为 n=q×pα=ϕ(q)×ϕ(pα)n=q\times p^{\alpha}=\phi(q)\times\phi(p^{\alpha})n=q×pα=ϕ(q)×ϕ(pα) ,其中 qqq 不含 ppp 的因子 故： $$\phi(n)=\phi(qp^{\alpha})=\phi(q)\phi(p^{\alpha})=qp\alpha\prod^k_{i=1}(1-\frac1{p_i})=n*\prod^k_{i=1}(1-\frac1{p_i})$$ 公式显然依然成立

    # 欧拉定理

    # 概念

    若 nnn, aaa 为正整数，且它们互质，则 $$a^{\phi(n)}\equiv1(mod\quad n)$$

    显然：当 nnn 为质数时 an−1≡1(modn)a^{n-1}\equiv1(mod\quad n)an−1≡1(modn) 就是费马小定理

    # 应用 —— 逆元

    如果在模运算中有除法，同时模数不是质数，则此时不可以用费马小定理，可以用欧拉定理
    逆元就相当于把费马小定理中的质数一般化一下： aaa 的逆元 = aϕ(n)−1a^{\phi(n)-1}aϕ(n)−1
