    # 定义

    Sierpinski三角形 是一种分形，是处理相似集的例子。

    # 构造

    # 正三角

    将一个正三角形通过连接三条边上的中点分为相等的四部分，去掉中心那一部分，然后将其余三部分递归地重复这个操作进行构造。
    如图我们访问过的递归部分为黑色三角形。

    # 正方形

    将一个正方形通过连接四条边上的中点分为四部分，去掉右下角那一部分，其余三部分按相同方式递归构造。

    # 结论

    # Sierpinski三角形 与 杨辉三角
