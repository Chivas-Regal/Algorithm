    # 前置知识——欧几里得算法

    循环实现: 

    int gcd(int a, int b){
	int tmp;
	while(b){
		tmp = a % b;
		a = b;
		b = tmp;
	}
	return a;
}

    # 扩展欧几里得算法

    # 代码思路

    本质与欧几里得流程类似，都是个递归函数

    1. 设置递归出口，即上面说的“最后一步”，同时做出最后一步的东西：{x = 1, y = 0; return a;}

    2. 递归的本质是欧几里得，所以d = exgcd(b, a % b, x, y);

    3. 回溯，利用上面裴蜀定理证明过程的到的公式： x1=y0,y1=x0−⌊ab⌋y0x_1=y_0,\quad y_1=x_0-\left \lfloor \frac ab \right \rfloor y_0x​1​​=y​0​​,y​1​​=x​0​​−⌊​b​​a​​⌋y​0​​ 后续遍历(回溯)求 (x,y)(x,y)(x,y)

    4. 我们即然求出来了 (a,b)(a,b)(a,b) 那么就可以用它，在最后一步进行return d;，得到 (a,b)(a,b)(a,b)

    # 程序

    int ex_gcd(int a, int b, int &x, int &y){
	if(b == 0) { x = 1, y = 0;  return a; }//出口：最后一步的解
	int d = ex_gcd(b, a % b, x, y);
	
	//回溯出上一步的(x,y)
	int tmp = x;
	x = y;
	y = tmp - (a / b) * y;
	
	return d;
}
